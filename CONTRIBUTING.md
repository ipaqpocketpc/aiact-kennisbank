# Bijdragen aan de AI Act Kennisbank

Bedankt voor je interesse om bij te dragen!

## Hoe bijdragen?

### 1. Issues

Meld fouten of suggesties via GitHub Issues:
- **Fout in content** - Verkeerde informatie, verouderd
- **Nieuwe content** - Suggestie voor nieuw artikel

### 2. Pull Requests

1. Fork de repository
2. Maak een branch: `git checkout -b fix/artikel-naam`
3. Maak je wijzigingen
4. Commit: `git commit -m "fix: corrigeer artikel X"`
5. Push en open een PR

## Content Richtlijnen

### Bronvermelding
- **Elke claim moet een bron hebben**
- Gebruik officiele bronnen (EUR-Lex, AI Office)
- Link direct naar het relevante artikel

### Formaat

**Data bestanden (JSON):**
- Gebruik consistente key namen
- Voeg `lastUpdated` timestamp toe
- Voeg `source` URL toe
